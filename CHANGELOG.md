## [0.1.0] - 2024-01-15

### Added
- Initial release
- User authentication (email/password, Google OAuth)
- Event discovery and browsing
- Event registration and checkout (Stripe integration)
- User dashboard with preferences and payment history
- Admin panel with event management
- User management and financial reports
- Coupon management system
- Email confirmations (Resend integration)
- QR code generation for tickets
- PDF receipt downloads
- Responsive design (mobile, tablet, desktop)
- Complete design system implementation

### Security
- Row Level Security (RLS) policies in Supabase
- Role-based access control (user, organizer, admin)
- Protected routes with middleware
- Secure webhook signature verification
- Input validation with Zod schemas
